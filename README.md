# calculator-kotlin

# Бинарный калькулятор на Kotlin

Программа на Kotlin, типа калькулятора, вывод в двоичной системе.

---

## Как работает
- Сначала программа спрашивает ввод выражения.  
- Можно писать с пробелами 
- Поддерживаются операторы: +, -, *, /.  
- Если ввести stop, программа завершится.  
- Результат выводится в двоичной системе (можно потом изменить в коде на другую систему).

---

## Структура проекта
- src/main/kotlin/io/IO.kt — интерфейс для ввода/вывода.  
- src/main/kotlin/io/Console_Io.kt — класс, который просто читает строки и пишет их в консоль.  
- src/main/kotlin/utils/Conversions.kt — функция, которая переводит числа в 2 систему.  
- src/main/kotlin/Main.kt — основной файл.

---

## Пример работы

Введи выражения типа 12 + 5. Напишите stop чтобы выйти.

12 + 5

Результат в двоичной системе: 1111

7   *   3

Результат в двоичной системе: 10101

stop

Выход из программы.
